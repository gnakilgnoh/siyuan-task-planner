# 思源笔记任务规划插件

本插件旨在帮助用户在思源笔记中规划和管理任务，提高工作效率。

## 功能介绍
- 任务规划：根据用户设定的时间间隔，自动关闭长时间未使用的标签页。
- 任务管理：用户可以在思源笔记中创建、编辑和删除任务。
- 任务优先级：用户可以为任务设置优先级，以便更好地管理任务。
- 任务截止时间：用户可以为任务设置截止时间，以便及时完成任务。

## 开始使用
用户需要在思源笔记中安装并启用本插件。安装完成后，用户可以在插件设置中配置任务规划的时间间隔。
