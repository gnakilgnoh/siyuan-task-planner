# 任务规划器

这是一个为思源笔记设计的插件：任务规划器。

## 初衷

工作中多个项目同时进行的时候，经常记不清楚任务的优先级，也记不清楚任务的截止时间。之前使用过一段时间的滴答清单，对于它的日历视图的任务规划功能还是很认同的，但是它天然不适合作为一个笔记系统，因此，我萌生了在思源笔记中实现一个类似的任务管理插件的想法。如果这个插件成功运行在集市中的话，那么之前的一些小的需求也可以通过插件的形式来实现了。

## 归纳需求

以下是我个人的需求, 后续会根据实际使用情况进行调整。

- 任务列表管理功能
  - 支持任务的创建、删除、优先级管理
- 日历功能
  - 月视图
  - 周视图
  - 三日视图

## 功能

待我个人的初步需求实现后，会在此处进行功能描述。

## 操作方式

安装完成后，在顶部栏右侧会出现插件对应的图标，点击后会建立一个页签，在页签中即可使用插件。
